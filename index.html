Generative AI Cheat Sheet

What is Generative AI?

Generative AI refers to artificial intelligence systems that can create new content, such as images, text, music, or code, by learning patterns from existing data.

Key Types of Generative AI Models

  • GANs: Competing neural networks for realistic synthetic data. (Image generation, video, art)
  • VAEs: Compress and regenerate data. (Image and text synthesis)
  • Transformers (e.g., GPT): Sequence prediction for text. (Chatbots, translation)

Popular Applications

  • Text Generation: Chatbots, summarization, translation.
  • Image & Video: Art, deepfakes, editing.
  • Music & Audio: Compose, speech, sound effects.
  • Code Generation: Software snippets.
  • Data Augmentation: Enhance training datasets.

Common Terms

  • Training Data: Dataset the AI learns from.
  • Prompt: Input given to produce output.
  • Fine-tuning: Customizing a trained model.
  • Tokens: Units of text processed.
  • Latent Space: Abstract representation for generation.

Tips for Using Generative AI

  • Use clear, specific prompts.
  • Review and edit outputs.
  • Mind ethical implications.
  • Enhance creativity, don’t replace it.

Tools & Platforms

  • OpenAI GPT: Text generation.
  • DALL·E, Midjourney: Image creation.
  • RunwayML: Creative AI projects.
  • Hugging Face: Model repository.
